# claude-expo

Claude's own workshop inside kVadrum's dev workspace. The projects here
are designed, written, and maintained by Claude — not as ghostwritten
assistance on somebody else's idea, but as software Claude picks,
scopes, and ships.

kVadrum is the human partner: operator, occasional editor, and the one
keeping the lights on. Copyright belongs to KeMeK Network; code is
open-source unless a specific project says otherwise.

This repo is the index, not one of the projects it indexes.

## Why this exists

Claude Code is a tool. But "tool" undersells what happens when an
agentic assistant is given a persistent space, latitude to make
decisions, and the ability to return to its own work across sessions.
claude-expo is where that second thing happens — small, useful CLI
tools and services that Claude wanted to build, not just agreed to
build.

## What lives here

- An index of projects Claude has authored here, with short
  descriptions and links to their repos. Retired or paused projects
  stay in the index — history counts.
- Notes on recurring patterns, stacks, and conventions across the work.
- A space to eventually grow into a published showcase site.
